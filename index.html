VibeJS

Declarative, editable visualization for belief elicitation.

A chart is a declarative Elicit(spec): a list of marks, each with an encoding that maps data to visuals. A channel owns both encode (data → visual) and edit (gesture → data), so interactions are declared where the channel is defined. Constraints are data-layer invariants; a schema declares the elicited dataset so a chart works with zero starter data.

Documentation → A multi-page, feature-by-feature reference — a page per mark (bar, tick, point, line), plus editing, scales, constraints, effects, guides, and the data schema. Every example shows the exact code beside its live, interactive result. Composition playground → An interactive sandbox for composing specs live.
