VibeJS — the new edit API

A channel owns both encode (data→visual) and edit (gesture→data). Interactions are declared where the channel is defined; a constraint on an edit governs that edit's channel by construction.

One bidirectional model: change-edits (drag / resize / cycle / custom), create as its own edit, pick: "nearest" proximity targeting, per-edit when arbitration, field-correct constraints, and edits that self-draw their guide via guide: true (no target).

1 · Vertical bars — drag to set the value

y is a linear channel carrying edit: drag(). Dragging a bar sets its y.

2 · Bars — maintainSum(100), field-correct

The sum constraint sits on the y edit with no explicit field — it defaults to that channel. Bars grow until the total hits 100. The dashed line is a declarative guide.

3 · Scatter — 2D move

Both x and y carry edit: drag(), so a drag moves the point in both.

4 · One mark, three edits (the payoff)

Drag to move (x+y), Shift-drag to resize (size→mag), click to recolor (cycle the group). Three channel-scoped edits on one mark, arbitrated by when.

5 · Likert — drag a point, or click an empty slot to place one

1D band x: edit: drag() snaps the dot to the nearest category, and a mark-level edit.create() lets a click on an empty slot mint a response (same scale inverse, one new datum).

6 · Custom edit — the escape hatch

edit.custom((datum, event) => …) gets the whole datum + event and returns a new datum. Here a vertical drag sets y and stamps touched.

7 · Create — click empty space to add a point

edit.create({ defaults }) is its own declaration: a plane click inverts the pointer through the positional channels (the same inverse drag uses) and appends a datum. Created points are then draggable through the channel edits. count(max) caps how many.

8 · Nearest pick + self-drawn guide

Thin bars are hard to hit, so the y edit uses pick: "nearest" — grab a bar from anywhere in its column. guide: true makes the edit draw its own snap ring and its clamp bounds; no target, no separate guide declaration.
