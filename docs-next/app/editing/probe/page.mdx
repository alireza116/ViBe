# Probe

<Lead>The **hover / click** flow, with no drag: the pointer **probes** a value — the mark follows the cursor as an uncommitted preview — and a click **settles** it. Give any edit `pick: "probe"` and it works this way, so the same primitive drives a correlation line, a slider knob, a Likert answer and a dot plot's tentative token. Add a `stage` and each click commits that stage's field and advances to the next — "set the line, now open the cone".</Lead>
