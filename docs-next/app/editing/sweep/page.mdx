# Sweep (you-draw-it)

<Lead>`edit.line.sweep()` is a drag that repaints each point’s value as the pointer crosses its column — the NYT “you draw it” interaction. It is sugar over `drag({ pick: "sweep", guide: true })`, and it is series-scoped: it locks onto the nearest line at drag-start and paints only that one.</Lead>

<ApiReference entries={api} />

<Section id="sweep" title="Sweep a single line">

Press and drag horizontally; every point the pointer passes takes the pointer’s value.

<Section id="series" title="Series-scoped sweep">

With multiple lines, a sweep locks onto the nearest line at drag-start and paints only it — the others hold. Grouping comes from the stroke field.
