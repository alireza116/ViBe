# Widgets

<Lead>Opinionated **survey instruments** — option rings, cell grids, tracks — each a pure recipe over the core API. They add no interaction surface: the look lives entirely in the [guide](/guides) layer, and the behaviour is a normal mark + a normal edit on the [probe](/editing/probe) driver + a normal constraint. Every example below is shown twice — once as the widget, once as a **self-contained** plain-API twin that draws the same look with inline `guides.custom` helpers, so you can see every line to change. Each factory returns an **ElicitSpec**: render with `Elicit(widgets.likert({…}))`.</Lead>
