# Interaction effects

<Lead>Feedback for interaction state (grabbing, proximity-selecting) lives in its own layer, never on a mark’s paint channels — so it can’t clobber your fill/stroke. Customize it through `effects`: a `grab` filter while dragging, and a `select` overlay (ring + highlight) for the proximity target.</Lead>

<ApiReference entries={api} />

<Section id="default" title="The default select effect">

A pick: "nearest" drag. The select effect draws the snap-zone ring and a highlight outline around the targeted dot — without touching the stroke you set.
