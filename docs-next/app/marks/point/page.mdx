# Point

<Lead>`point` is the channel-driven scatter mark: every channel — x, y, size, fill, stroke, angle — resolves through the global scales. Default shape is a circle; `shape: 'square'` emits a centred rect (side `2 × size`) that can rotate about its centre.</Lead>

<Section id="angle" title="Rotating squares and ticks">

An `angle` channel orients a mark about its centre. Degrees are math convention (0° = +x, counterclockwise); `rotate()` inverts the pointer through the same scale. Squares use `point({ shape: 'square' })`; short `tickX` / `tickY` segments (with `length`) rotate the same way — useful for directional markers on a scatter.
