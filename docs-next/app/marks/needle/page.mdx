# Needle

<Lead>A pivoted **needle** (NYT-style gauge or software dial). It encodes a value on the `angle` channel — degrees via the channel’s scale — and draws a tapered pointer + hub. The default semicircle is **horizontal** (left → right through the top, like a speedometer). Use `orient` (`top`·`right`·`bottom`·`left`), `arc: "full"`, or explicit `start`/`end` — and keep `scale.range` in sync. Optional `x`/`y` place the pivot on categorical or linear axes (many small needles in one chart).</Lead>

<ApiReference entries={api} />
