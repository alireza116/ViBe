# Axis, Grid & Rule

<Lead>Axes and gridlines are composable marks. Configure them the easy way through the global `axes: {…}` convenience (ticks, formats, titles, hiding, free positioning), or add `axisX` / `axisY` / `gridY` / `ruleY` as explicit features for full control.</Lead>

<ApiReference entries={api} />

<Section id="global" title={"The global axes:{} convenience"}>

Pass a per-channel config to shape ticks, formats and titles, or a transform to reposition an axis. axes: false drops them entirely.
