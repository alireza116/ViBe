<Section id="edit" title="Editable slice boundaries">

Wire `edit.arc.edge()` and **every** boundary gets a draggable handle — including the seam where the last slice meets the first on a full circle, so _n_ slices give _n_ handles. Dragging a boundary moves value between exactly the two slices it separates: one grows by what the other loses, so the total is preserved.
