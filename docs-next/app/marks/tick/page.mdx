# Tick

<Lead>A tick is bar’s zero-thickness sibling: a thin line marking a value on the linear axis and spanning the other axis. Over a category band it reads as a value marker (like a bar without the fill); over a full continuous extent it reads as a rug / strip. `inset` shrinks each end, `length` fixes a centered length.</Lead>

<ApiReference entries={api} />

`tickY` marks a `y` value spanning each category's band — the same channel/edit model as a `bar`, so it drags identically.
