# Text

<Lead>A per-datum **label** (Observable Plot’s text). It places a string at a positional (x, y); `text`/`fontSize`/`textAnchor`/`lineAnchor`/`dx`/`dy` are read RAW (a field or a constant), `angle` is degrees, and `format` is a d3-format string or function (display-only). A text mark is editable like any other: drag to reposition, drag a value to update a numeric readout, `cycle()` a label, `rotate()` it, or `editText()` to retype its content.</Lead>
