# Stacked dots

<Lead>A **dot histogram** — the "drop tokens into slots" elicitation. Each datum is one token; tokens sharing a slot stack into a countable column. Click an empty slot to add a token (`create`), click a token to remove it (`remove`). The belief is just how many tokens sit in each slot — `data.filter(d => d.bin === b).length`.</Lead>

<ApiReference entries={api} />
