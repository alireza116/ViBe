# What you can elicit

<Lead>
  VibeJS lets you build charts you can use to elicit information (beliefs,
  preferences, and other things) from people. drag a bar, place a point, turn a
  dial — and you get structured data back (see the panel next to each chart).
  Try the questions below.
</Lead>

<Section id="allocate" title="How would you allocate 100% across these categories?" codeMode="collapsed">

Use a bar chart when the pieces have to add up. Drag any bar; the others adjust so the total stays 100.

<Example
  code={allocateABudget}
  title="Allocate a budget"
  blurb="Drag the bars to split 100% across categories. Raising one lowers the others."
  try={
    <>
      <b>Drag</b> any bar. The shares always add up to 100.
    </>
  }
/>

</Section>

<Section id="scatter" title="Where do people sit on age vs immigration attitude?" codeMode="collapsed">

Use a scatter plot to ask a two-part question. The chart starts empty — double-click to add someone, then drag them into place.

<Example
  code={placeBeliefs}
  title="Age vs immigration attitude"
  blurb="An empty scatter plot. Double-click to place people; drag to move them. Colour runs from red (anti) to green (pro)."
  try={
    <>
      <b>Double-click</b> to add a person · <b>drag</b> to move them.
    </>
  }
/>

</Section>

<Section id="forecast" title="What trend do you expect over time?" codeMode="collapsed">

Use a line chart when you want someone to draw a path. Drag across the chart to sketch the forecast you have in mind.

<Example
  code={drawAForecast}
  title="Draw the trend you expect"
  blurb="Sketch the line with your mouse — like drawing a forecast by hand."
  try={
    <>
      <b>Drag</b> left-to-right across the chart to reshape the line.
    </>
  }
/>

</Section>

<Section id="needle" title="How likely is Candidate D to win?" codeMode="collapsed">

Use a needle dial for a single choice on a scale — from “very likely D” to “very likely R.” Drag the needle to answer.

<Example
  code={likelihoodNeedle}
  title="Likelihood gauge"
  blurb="A dial you can turn — drag the needle to pick how likely something feels."
  try={
    <>
      <b>Drag</b> the needle left or right.
    </>
  }
/>

</Section>

<Section id="geo" title="Where do you think is the best place to live in Vancouver?" codeMode="collapsed">

Place pins on a map to answer. Click to drop a pin; drag one to move it.

<Example
  code={pinsOnAMap}
  title="Best place to live"
  blurb="Click the map to drop pins for places you’d live; drag a pin to fine-tune."
  try={
    <>
      <b>Click</b> to place a pin · <b>drag</b> an existing one.
    </>
  }
/>

</Section>

<Section id="survey" title="How easy is this tool to use?" codeMode="collapsed">

Classic survey scales work too. Hover to preview, then click to lock in an answer.

<Example
  code={answerAScale}
  title="Answer a survey scale"
  blurb="A familiar agree/disagree question — hover, then click your answer."
  try={
    <>
      Hover across the scale, then <b>click</b> your answer.
    </>
  }
/>

</Section>
